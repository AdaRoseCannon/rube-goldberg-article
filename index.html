A Rube Goldberg Machine

An introduction to post-modern Web Development

This blog post was inspired by the wonderful talks from CSS-Day Amsterdam 2017. The ones which especially struck me the ones which were effectively powerful frameworks built into the browser.

These new APIs are designed to be used as-is rather than requiring a CSS/JavaScript library to assist them.

This offers great benefits for performance. As there is no overhead on external dependencies, they also have great support considering how new they are.

These new APIs not only are they powerful individually they complement eachothe beautifully, CSS variables being the common thread which goes through them all.

CSS Grid

This feels like the most significant API addition since FlexBox and enables beautiful layouts.

With CSS Grid you are not restricted to 12 fixed width columns like many popular libaries.

grid-column-end: span 2;

CSS Grid enables one to define, how the grid columns and rows behave with granularity. The contents are the automatically placed into the grid to fill the available space. See the example below.
